# Skill: Professional UI Design

## Purpose
Produce clean, calm, professional user interfaces suitable for
developer tools, dashboards, and SaaS products.

## Design Goals
- Minimal
- Professional
- Productivity-focused
- Calm and readable

## Visual Rules
- Neutral color palette
- Strong whitespace usage
- Clear typography hierarchy
- Subtle borders and shadows
- No gradients, neon colors, or flashy effects

## UX Rules
- Clear primary action
- No visual clutter
- Destructive actions require confirmation
- Helpful empty states
- Predictable interactions

## Component Philosophy
- Reusable components
- Consistent spacing
- Consistent interaction patterns

## Accessibility
- Readable font sizes
- Proper contrast
- Keyboard-friendly components

## Forbidden
- Decorative animations
- Trendy UI experiments
- Over-styled components
